Project Title

Explore Chapel Hill Outdoors

Project Description

Our website aims to be a user-friendly tool for finding nearby outdoor activities in Chapel Hill, mainly hiking, biking, camping, and disc-golfing (the finality of these 4 options is up for debate). The user will navigate through a javascript-enabled aesthetically pleasing home page in order to select a hike, bike trail, camping space, or disc-golf course (There will be numerous pages for each kind of activity). This will load the information page (see mock-up below), which will display information on the chosen activity. There will only be one design for the information page, but the information displayed will be different depending on the specific hike, etc. chosen. This will be accomplished with a back-end database containing information on each activity. The information page contains pictures (maybe), a map loaded by the Google API, information, a twitter feed including any hashtag mention of that hike/course/etc, links to other activities the user may like, and a of course a link back to the home page. We still don't know what other features will be included in the site, but this is the basic idea. We believe that this adequately meets the requirements for the final project, but aren't totally positive.
